# Spectra Render
 An unbiased, hobby pathtracer

## Features
 - ✅ Spectrum utility functions
 - ⬜ Materials :
 	- ✅ Color
    - ✅ Emission
 	- ⬜ BRDF
 	- ⬜ Roughness
 	- ⬜ Metallic
 - ⬜ BRDFs supported :
 	- ✅ Lambertian
 	- ⬜ Trowbridge-Reitz (GGX) 
 	- ⬜ Torrance-Sparrow
 - ✅ Ray-Sphere intersection
 - ⬜ Ray-Triangle intersection
 - 
 - ⬜ More to come ...
